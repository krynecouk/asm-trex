# asm-trex

T-Rex can't even assembly for Atari 2600 (MOS 6507).

![Screenshot of playfield](screenshot.png)

## Prerequisites

- [CMake](https://cmake.org/)
- [dasm](https://dasm-assembler.github.io/)
- [Stella](https://stella-emu.github.io/)

## Compilation

```cmake
make
```

## Execution

```cmake
make run
```
